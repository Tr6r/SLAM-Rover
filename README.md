## 🎯 Objectives of the Project
### Develop a robot capable of real-time self-localization and 2D mapping based on and extended from the NOX project architecture.
## ✨ Key Features
- **2D Mapping:** Utilizes a LiDAR sensor to process laser scan data and generate a real-time 2D map of the environment.
- **Localization:** Uses a 2-channel encoder to estimate odometry and improve real-time positioning accuracy.
- **Visualization:** Integrates with RViz to visualize the generated map and monitor the robot's current pose in real time.
