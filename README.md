## 🎯 Objectives of the Project
### Develop a robot capable of self-localizing on a 2D map in real-time.
## ✨ Key Features
- **2D Mapping:** Utilizes a LiDAR sensor to process laser scans and construct a 2D map of the environment.
- **Localization:** Employs a 2-channel encoder to calculate odometry (distance traveled) for precise real-time positioning.
- **Visualization:** Uses RViz to display the generated 2D map and track the robot's current pose.
